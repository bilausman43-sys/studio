# **App Name**: Aviator Insights

## Core Features:

- Game Round Data Input: Allow users to manually record or input historical Aviator game round data, such as crash multipliers and flight durations.
- Historical Pattern Analysis (AI Tool): An AI tool that analyzes collected historical game data to identify statistical trends, frequently occurring multipliers, and common 'safe zones' for strategy development, rather than predicting specific outcomes.
- Probabilistic Crash Indicator: Visually represents the probability of the airplane crashing at various multipliers, based on the analyzed historical data, to help users make more informed decisions.
- Betting Strategy Simulator: A simulation environment where users can test hypothetical betting strategies against collected historical data or a probabilistic model, allowing for risk-free experimentation.
- User's Game History & Stats: Securely store and display a user's personal history of game rounds and simulated bets, along with performance statistics, to track their progress and learning.
- Royal.Casino Quick Launch: A direct button or link that allows users to quickly open the royal.casino app or website to place their bets manually, based on insights from 'Aviator Insights'.

## Style Guidelines:

- Color Palette Rationale: A dark theme is chosen to convey a sense of sophistication and intensity, typical of analytics and gaming interfaces. A rich, deep indigo primary color provides a regal and trustworthy feel. The background is a desaturated version of this, ensuring readability and focus. A vibrant blue accent offers high contrast for calls to action and important data, representing clarity and foresight.
- Primary Color: #321A6E (a deep, sophisticated indigo).
- Background Color: #22202C (a very dark grayish-purple, enhancing depth and focus).
- Accent Color: #52A0FF (a vibrant sky blue, for key actions and highlights).
- Headline Font: 'Space Grotesk' (sans-serif) for a modern, techy, and analytical feel in titles and prominent text.
- Body Font: 'Inter' (sans-serif) for clear, objective, and neutral readability of data, descriptions, and longer content.
- Utilize sleek, minimalist line icons that clearly represent data points, graphs, and gaming actions, aligning with the modern and analytical aesthetic.
- Employ a modular and structured layout with clear data visualization panels, ensuring that complex information is digestible and easy to navigate. Prioritize space and visual hierarchy.
- Implement subtle and smooth transition animations for data updates, chart rendering, and simulation results to enhance user engagement without causing distraction, reflecting professionalism.
